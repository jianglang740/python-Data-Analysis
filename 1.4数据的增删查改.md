# 1.4.1增加数据

* 先创建示例dataframe：
```python
import pandas as pd
#解决数据输出时列名对不齐的问题
pd.set_option("display.unicode.east_asian_width",True)
data = [[110,105,99],[105,88,115],[109,120,130],[112,115,140]]
name = ["明日","七月流火","高媛媛","二月二"]
columns = ['语文','数学','英语']
df = pd.DataFrame(data = data,index = name,columns = columns)
print(df)
```
结果：
```
          语文  数学  英语
明日       110   105    99
七月流火   105    88   115
高媛媛     109   120   130
二月二     112   115   140
```

* 1.直接给dataframe对象赋值：

* 2.使用loc方法在dataframe的最后添加一列物理成绩：
```python
import pandas as pd
#解决数据输出时列名对不齐的问题
pd.set_option("display.unicode.east_asian_width",True)
data = [[110,105,99],[105,88,115],[109,120,130],[112,115,140]]
name = ["明日","七月流火","高媛媛","二月二"]
columns = ['语文','数学','英语']
df = pd.DataFrame(data = data,index = name,columns = columns)
df.loc[:,"物理"] = [80,79,60,50]
print(df)

:

         语文  数学  英语  物理
明日       110   105    99    80
七月流火   105    88   115    79
高媛媛     109   120   130    60
二月二     112   115   140    50
```

* 使用insert（）方法在指定位置插入一列数据：

（在语文的后面新加一列物理成绩）
```python
import pandas as pd
#解决数据输出时列名对不齐的问题
pd.set_option("display.unicode.east_asian_width",True)
data = [[110,105,99],[105,88,115],[109,120,130],[112,115,140]]
name = ["明日","七月流火","高媛媛","二月二"]
columns = ['语文','数学','英语']
df = pd.DataFrame(data = data,index = name,columns = columns)
wl = [80,79,60,50]
df.insert(1,"物理",wl)
print（df）

:

          语文  物理  数学  英语
明日       110    80   105    99
七月流火   105    79    88   115
高媛媛     109    60   120   130
二月二     112    50   115   140
```
